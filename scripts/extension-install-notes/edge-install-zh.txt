KeyNest — Microsoft Edge 扩展安装说明
========================================

适用文件：KeyNest-Edge.zip

一、安装步骤
1. 解压 zip，得到名为 KeyNest 的文件夹（请勿只复制零散文件）。
2. 在 Edge 地址栏输入 edge://extensions 并回车。
3. 打开左下角或侧栏「开发人员模式」。
4. 点击「加载解压缩的扩展」，选中解压后的 KeyNest 文件夹。
5. 打开 KeyNest 桌面端并解锁，在窗口底部勾选「允许浏览器扩展连接本机端口 17373」。

二、更新扩展
在 edge://extensions 页面点击 KeyNest 助手旁的「重新加载」；
或移除旧版后，按「一」重新加载新版文件夹。

三、使用说明
- 含密码框的登录页右下角会出现「KeyNest 填入」；也可点击工具栏扩展图标填入。
- 登录成功后可提示是否保存到本机保管库（需桌面端已解锁且开启桥接）。
- 账号密码仅在本机读写，不会上传到互联网。
