# 参与者公约

## 我们的承诺

为营造开放、友善的环境，我们作为贡献者与维护者承诺：无论年龄、体型、身体健全与否、民族、性征、性别认同与表现、经验、国籍、外貌、宗教或性取向，参与本项目的每个人都应免于骚扰。

## 我们的准则

有助于创造积极环境的行为包括：

- 使用友善、包容的语言
- 尊重不同的观点与经验
- 得体地接受建设性批评
- 将社区利益置于个人利益之上
- 对其他社区成员保持同理心

不可接受的行为包括：

- 性暗示用语或意象、不受欢迎的性关注或示好
- 挑衅、侮辱/贬损的评论、人身或政治攻击
- 公开或私下骚扰
- 未经明确许可公开他人隐私信息（如住址、电子邮箱）
- 其他在职业场合通常被认为不适当的行为

## 我们的责任

维护者负责阐明可接受行为的标准，并对不当行为采取恰当、公平的纠正措施。

## 适用范围

本公约适用于所有本项目的公共空间，例如 GitHub 仓库、Issue、拉取请求与面向社区的讨论区。若个人在公共或私人场合的行为严重威胁社区正向氛围，维护者可追加限制。

## 执行

可通过 GitHub 的 Issue/私信向维护者报告骚扰或不可接受行为。维护者将审查并调查所有投诉，并以认为必要且适当的方式回应，可能包括警告、临时禁言或永久禁止参与本项目。

## 致谢

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org)，版本 2.1。
